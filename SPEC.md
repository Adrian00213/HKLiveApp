# HKLive - 香港即時生活平台

## 1. Concept & Vision

**HKLive** 係一個為香港人而設嘅即時生活資訊平台，以「活在當下」為核心概念。香港人生活節奏快，需要第一時間知道附近發生咩事——無論係交通意外、優惠折扣、演唱會、還是社區討論。平台嘅最大特色係「超本地化」同埋「即時性」，所有資訊都根據用戶位置實時更新。

**Tagline**: 「香港，一触即发」

---

## 2. 香港市場痛點分析

### 現有問題
| 痛點 | 現有解決方案 | 問題 |
|------|-------------|------|
| MTR/巴士/小巴到站時間 | 各App分散 | 冇統一體驗 |
| 路面交通實況 | 新聞/其他地方 | 不夠即時 |
| 空氣質素 | AQHI網站 | 不夠視覺化 |
| 優惠資訊 | OpenRice/其他地方 | 分散 |
| 演唱會/活動資訊 | 的城市票務/各場地 | 分散，冇地圖 |
| 社區討論 | Facebook群組 | 冇地理位置感 |
| 突發事件 | 新聞App | 被動接收 |

### HKLive 解決方案
將所有即時資訊整合到單一平台，以地圖為核心介面

## 4. 技術架構

### 前端
- **Framework**: React Native (Expo)
- **地圖**: MapLibre GL JS
- **即時**: Socket.IO Client
- **導航**: @react-navigation/bottom-tabs
- **狀態**: Zustand
- **多媒體**: expo-image-picker, expo-av（語音/影片）

### 後端
- **Runtime**: Node.js + Express
- **即時**: Socket.IO
- **數據庫**: Firebase Firestore
- **認證**: Firebase Auth
- **推送**: Firebase Cloud Messaging

### API 設計
```
POST   /api/auth/register
GET    /api/map/incidents?lat=&lng=&radius=
POST   /api/map/report-incident（支援圖片/語音）
GET    /api/deals?lat=&lng=
GET    /api/events?lat=&lng=&date=
GET    /api/discussions?district=
POST   /api/discussions（支援多媒體）
POST   /api/discussions/:id/media（上傳圖片/影片/語音）
GET    /api/weather/aqhi
GET    /api/transport/mtr-status
GET    /api/transport/bus-arrival?stop_id=
```

## 6. 討論區多媒體功能（詳細設計）

### 支援格式
| 類型 | 格式 | 大小限制 |
|------|------|---------|
| 圖片 | JPG/PNG/GIF | 10MB |
| 影片 | MP4/MOV | 50MB |
| 語音 | M4A/MP3/WAV | 10MB |

### 發布流程
1. 選擇類型（文字/圖文/影片/語音）
2. 輸入內容
3. 附加媒體（可選）
4. 選擇地區（自動定位）
5. 設定匿名/公開
6. 發布

### 媒體展示
- 圖片: 九宮格縮圖，點擊放大
- 影片: 內嵌播放，支援全屏
- 語音: 波形播放器，自動轉文字

## 7. 自動化功能

### 智能推送
- 📍 到達某區 → 推送該區優惠
- ⏰ 活動開始前1小時 → 提醒
- 🚨 附近有交通意外 → 繞路建議
- 🌧️ 天氣突變 → 即時警告

### 自動更新
- 📍 用戶移動 → 更新附近資訊
- 🔄 背景自動刷新資訊（5分鐘）
- 📱 下拉刷新 → 即時更新

### 數據同步
- ☁️ 雲端同步收藏/設置
- 📤 跨設備登入 → 數據同步
